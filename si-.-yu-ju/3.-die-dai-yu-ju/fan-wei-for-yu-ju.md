# 范围for语句

```
for (declaration : expression) {
    statement
}
```

* 语法
  1. 首先范围for循环的条件表达式与普通的for循环条件表达式有些区别, 范围for循环的条件表达式只有一个表达式, 包括两个变量和一个运算符
  2. expression是一个对象, 用于表示一个序列, 如字符串, 向量等
  3. declaration部分负责定义一个变量(循环控制变量), 该变量将被用于访问序列中的基础元素, 如: `auto i`, 一般使用auto来让编译器决定变量i的类型, 如果明确知道变量i需要的类型, 也可以使用, 如序列是一个字符串时, 就可以将auto换为char, 不过一般不用具体化
* 效果
  1. 循环控制变量从序列的第一个基础元素开始, 被赋值为序列的第一个基础元素
  2. 执行完循环体后, 如果序列的下一个基础元素存在, 则循环控制变量被赋值为序列的下一个基础元素
  3. 如果序列不存在下一个基础元素, 即循环到达终点, 控制跳出循环
* 注意
  1. 如果需要修改序列中的元素, 则需要将循环控制变量定义为引用, 如: `auto &i`, 否则无法改变序列中的元素, 因为变量与序列中的元素是相互独立的
