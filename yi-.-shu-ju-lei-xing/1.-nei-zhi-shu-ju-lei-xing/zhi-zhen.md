# 指针

* 空指针常量: nullptr, 0, NULL
* void\*指针 用法与C语言相同, 这里主要介绍其不同点:
  1.  C++只有使用显式强制类型转换才能把指向void的指针赋给其他类型的指针, 如:

      ```
      int ar[5] = {4, 5, 6, 7, 8};
      int *pi;
      void *pv = ar;
      pi = pv;        // C可以, 但C++不行
      pi = (int *)pv; // C和C++都可以
      ```
  2. C++可以把派生类对象的地址赋给基类指针, 但在C中没有这里涉及的特性
