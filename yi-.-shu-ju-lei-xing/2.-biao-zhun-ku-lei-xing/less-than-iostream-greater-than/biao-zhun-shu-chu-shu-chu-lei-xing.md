# 标准输出输出类型

### 一. 头文件: \<iostream>

C++虽然能兼容C语言的标准输入输出头文件\<stdio.h>, 但对于C++本身而言, 它并未定义任何的输入输出语句, 取而代之, 包含了一个全面的标准库来提供IO机制(以及很多其他设施).&#x20;

**iostream**库包含两个基础类型**istream**和**ostream**, 分别表示输入流和输出流.

标准库定义了4个IO对象:

**1**个**istream**类型对象: **cin**

**3**个**ostream**类型对象: **cout**, **cerr**, **clog**

### 二. 主要对象使用方法

#### 1. cin

*   基本使用方法:

    ```
    int n1, n2;
    std::cin >> n1 >> n2;
    ```

    1. 首先4个IO对象都是定义在命名空间"std"中, 所以需要声明命名空间
    2. 输入操作需要通过输入运算符">>"进行
    3. 读取标准输入的顺序是从前到后, 即先将标准输入保存到cin, 然后通过输入运算符赋给n1, 然后(std::cin >> n1)这一整块返回前面的部分, 即std::cin, 所以读取完第一个值之后, n1被赋值, 输入语句等价于`std::cin >> n2;`, 然后重复读取n1时的过程
* 注意事项:
  1. cin的读取忽略第一个非空白字符前的所有空白字符
  2. 读取从第一个非空白字符开始, 直到下一个空白字符出现
  3. 输入类型和变量定义类型要一致

#### 2. cout

*   基本使用方法:

    ```
    std::cout << "The sum of" << n1 << " and " << n2 << " is " << v1 + v2 << std::endl;
    ```

    1. 与cin相同, 需要声明命名空间
    2. 输出操作通过输出运算符"<<"进行
    3. 标准输出的顺序是从最前面两个量开始, 该例中, 首先是`std::cout`和`"The sum of"`这两个量结合, 即(std::cout << "The sum of"), 程序将字符串"The sum of"保存到cout, cout连接标准输出, 显示到屏幕上; 然后该表达式的运算结果是左边的值, 即std::cout, 然后std::cout又与后面的值n1结合, 依次类推, 最后将后续的字符串打印完
    4. `std::endl`是一个被称为**操纵符**的特殊**值**, 它的效果是结束当前行, 并将与设备关联的缓冲区中的内容刷到设备中. 前一个效果与转义字符相同, 而后一个效果则是保证到目前为止程序产生的所有输出都真正写在输出流中, 而不是仅停留在内存中等待写入流.
* 注意事项:
  1. 字符串字面量和变量需要用两个输出运算符连接, 不能放在同一个输出运算符的右边
  2. `std::endl`应该尽量添加, 否则, 如果程序崩溃, 留在缓冲区里的输出会导致关于程序崩溃位置的错误推断

### 三. 重载运算符

#### 1. 输入运算法: >>

* 效果: 接受一个istream作为左侧运算对象, 接受一个对象作为其右侧运算对象, 从给定的istream读入数据, 并存入到给定对象中
* 特点:
  1. 可连接, 如: `cin >> i >> j;`
  2. 每次只进行**一个**运算符两侧对象的运算
  3. 每次运算的返回值为左侧对象

#### 2. 输出运算符: <<

* 效果: 左侧的运算对象必须是一个ostream对象, 右侧的运算对象是要打印的值, 此运算符将给定的值写到给定的ostream对象中
* 特点:
  1. 可连接, 如: `cout << "hi" << "bye";`
  2. 每次只进行**一个**运算符两侧对象的运算
  3. 每次运算的返回值为左侧对象
