# 字符串类型string

标准库类型**string**表示可变长的字符序列, 使用string类型必须首先包含string头文件. 作为标准库的一部分, string定义在命名空间std中.

C++标准库中所定义的字符串类型比用数组实现的字符串更加方便与灵活一些, 以下是string类型的用法

### 一. 创建

```
string s1;        // 默认初始化, s1是一个空串
string s2 = s1;    // s2是s1的副本
string s3 = "hiya";    // s3是该字符串字面值的副本
string s4(10,'c');    // s4的内容是cccccccccc
```

需要注意的是, C++风格的字符串是没有末尾的空字符的, 但字符串字面值是有末尾的空字符的

* 初始化string对象的方式

|           方式          |                 效果                 |
| :-------------------: | :--------------------------------: |
|      `string s1`      |           默认初始化, s1是一个空串           |
|    `string s2(s1)`    |              s2是s1的副本              |
|    `string s2 = s1`   |    等价于`string s2(s1)`, s2是s1的副本    |
|  `string s3("value")` |  s3是字面值"value"的副本, 除了字面值最后的那个空字符外  |
| `string s3 = "value"` | 等价于`s3("value")`, s3是字面值"value"的副本 |
|  `string s4(n, 'c')`  |         把s4初始化为连续n个字符c组成的串         |

使用等号的初始化实际上执行的是拷贝初始化, 即把等号右侧的初始值拷贝到新创建的对象中去.

不使用等号的初始化执行的是直接初始化

当要初始化的值有多个时, 一般用直接初始化, 否则需要显式创建临时对象来进行拷贝
